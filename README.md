Sublime-Nitrous-Theme
=====================

This is nitrous IDE theme for Sublime Text

#### How to install

1. Browse Packages ...
2. Copy Sublime Text Theme file to this directory
3. Back your Sublime Text, change current Color Scheme to Nitrous and enjoy new scheme
